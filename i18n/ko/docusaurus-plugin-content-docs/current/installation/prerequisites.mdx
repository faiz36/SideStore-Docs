SideStore의 설치 과정은 복잡해 보일 수 있지만 단 ***한번*** 만 컴퓨터를 사용하고 곧 기기에서 SideStore를 사용할 수 있습니다!

## 필요 조건

SideStore를 설치하기 위해선:
- iOS 또는 iPadOS 14 이상을 실행중인 iOS 또는 iPadOS 기기.
- Windows, macOS, 또는 Linux 컴퓨터 (Linux 지원은 실험적입니다).
- Apple ID.
- StosVPN 앱.
- 약간의 시간 (과정은 빠르게 넘어가지만 문제 해결을 위해선 시간이 필요합니다).

iOS/iPadOS 16 이상 사용자는 *무조건*  개발자 모드가 활성화되어 있어야합니다.
방법은:
1. 설정 앱을 엽니다.
2. '개인정보 보호 및 보안'으로 들어갑니다.
3. 밑에 있는 '개발자 모드'를 활성화합니다.

> **주의:** SideStore를 성공적으로 설치 했을때만 '개발자 모드' 옵션이 보입니다.

## StosVPN

SideStore는 장치 내 VPN을 사용하여 SideStore가 내부 서비스와 통신할 수 있도록 합니다.

1. [공식 앱스토어](https://apps.apple.com/us/app/stosvpn/id6744003051)에서 StosVPN을 설치하세요.
2. VPN을 활성화하세요.

이 VPN은 SideStore에서 앱을 설치하거나 업데이트할때 켜져 있어야 합니다.
